# <a name="delete-a-database-diagram-visual-database-tools"></a>Eliminazione di un diagramma di database (Visual Database Tools)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
Quando un diagramma di database non serve più, è possibile eliminarlo.  
  
### <a name="to-delete-a-database-diagram"></a>Per eliminare un diagramma di database  
  
1.  In Esplora oggetti espandere la cartella **Diagrammi di database** .  
  
2.  Fare clic con il pulsante destro del mouse sul diagramma di database che si desidera eliminare.  
  
3.  Scegliere **Elimina** dal menu di scelta rapida.  
  
4.  Un messaggio richiederà di confermare l'eliminazione. Scegliere **Sì**.  
  
Il diagramma di database verrà eliminato dal database. Quando si elimina un diagramma di database, le tabelle del diagramma non vengono eliminate.
