# <a name="the-sql-server-management-studio-environment"></a>Ambiente di SQL Server Management Studio
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
[!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] offre un ambiente completo per la gestione e lo sviluppo di query in [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)].  

> [!NOTE]
> Sono stati rilevati conflitti con i driver grafici che generano problemi nel processo di rendering di SQL Server Management Studio e pertanto tale accelerazione grafica hardware è stata disabilitata per impostazione predefinita.  Se si incontrano difficoltà di rendering o altri problemi visivi nell'interfaccia di SSMS, verificare che l'impostazione "Usa accelerazione grafica hardware se disponibile" sia *deselezionata* in Strumenti > Opzioni.
