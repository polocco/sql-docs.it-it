# <a name="reserved-database-names-in-parallel-data-warehouse"></a>Nomi di database riservati in parallelo data warehouse
Un nome di database riservato è un nome che non può essere assegnato a un database utente in SQL Server PDW.  
  
Elenco dei nomi di database riservati:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
