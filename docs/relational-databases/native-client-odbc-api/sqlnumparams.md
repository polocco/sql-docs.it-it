# <a name="sqlnumparams"></a>SQLNumParams
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  Miglioramenti nel motore di [!INCLUDE[ssSQL11](../../includes/sssql11-md.md)] database che iniziano con consentire SQLNumParams per ottenere descrizioni più accurate dei risultati previsti. Questi risultati più accurati possono differire dai valori restituiti da SQLNumParams nelle versioni precedenti di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
 Per altre informazioni, vedere [Metadata Discovery](../../relational-databases/native-client/features/metadata-discovery.md).
