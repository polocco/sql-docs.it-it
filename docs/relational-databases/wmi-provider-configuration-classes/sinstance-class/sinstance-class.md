# <a name="sinstance-class"></a>Classe SInstance
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  La [classe SInstance](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) rappresenta un'istanza di [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
 La [classe SInstance](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) può essere utilizzata per gestire protocolli e librerie di rete associati all'istanza di [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] . È inoltre possibile specificare un certificato di sicurezza per l'istanza di [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
