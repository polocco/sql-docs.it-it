# <a name="syspdw_diag_event_properties-transact-sql"></a>sys. pdw_diag_event_properties (Transact-SQL)
[!INCLUDE[tsql-appliesto-xxxxxx-xxxx-xxxx-pdw-md](../../includes/tsql-appliesto-xxxxxx-xxxx-xxxx-pdw-md.md)]

  Include informazioni sulle proprietà associate agli eventi di diagnostica.  
  
|Nome colonna|Tipo di dati|Descrizione|Range|  
|-----------------|---------------|-----------------|-----------|  
|**event_name**|**nvarchar(255)**|Nome dell'evento di diagnostica specifico.||  
|**property_name**|**nvarchar(255)**|Nome di una proprietà dell'evento.||
