# <a name="leveldepth-property-ado-md"></a>Proprietà LevelDepth (ADO MD)
Indica il numero di livelli tra la radice della gerarchia e un [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Valori restituiti  
 Restituisce un valore **Long** integer ed è di sola lettura.  
  
## <a name="remarks"></a>Osservazioni  
 Utilizzare la proprietà **LevelDepth** per determinare la distanza dell'oggetto [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md)dal livello radice della gerarchia. Il **LevelDepth**di un membro al livello radice è 0. Corrisponde alla proprietà [Depth](../../../ado/reference/ado-md-api/depth-property-ado-md.md) di un oggetto [Level](../../../ado/reference/ado-md-api/level-object-ado-md.md) .
