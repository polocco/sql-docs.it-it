# <a name="source-property-ado-md"></a>Proprietà Source (ADO MD)
Indica l'origine dei dati nel [celle](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Impostazioni e valori restituiti  
 Imposta o restituisce un **Variant**ed è in lettura/scrittura per gli oggetti del set di [celle](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) chiusi e in sola lettura per gli oggetti del set di **celle** aperti. La **variante** deve contenere una **stringa**valida, ad esempio una query MDX.
