# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
Specifica il numero di record da recuperare da un [Recordset](../../../ado/reference/ado-api/recordset-object-ado.md).  
  
|Costante|Valore|Descrizione|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|Recupera il resto dei record nel **Recordset**, dalla posizione corrente o da un segnalibro specificato dal parametro *Start* del metodo [GetRows](../../../ado/reference/ado-api/getrows-method-ado.md) .|
