PyODBC è un software supportato dalla community. Microsoft offre il proprio contribuito per la community open source di pyODBC ed è un partecipante attivo nel repository all'indirizzo [https://github.com/mkleehammer/pyodbc/](https://github.com/mkleehammer/pyodbc/). Per questo software non è tuttavia disponibile il supporto tecnico Microsoft.

Per ottenere assistenza, registrare un [problema nel repository GitHub di pyODBC](https://github.com/mkleehammer/pyodbc/issues) oppure usare altre risorse della community di Python.
