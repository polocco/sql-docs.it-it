# <a name="getlong-method-sqlservercallablestatement"></a>Metodo getLong (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come oggetto **long** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[getLong (int)](../../../connect/jdbc/reference/getlong-method-int.md)|Recupera il valore del parametro designato come oggetto **long** nel linguaggio di programmazione Java in base all'indice del parametro.|  
|[getLong (java.lang.String)](../../../connect/jdbc/reference/getlong-method-java-lang-string.md)|Recupera il valore del parametro designato come oggetto **long** nel linguaggio di programmazione Java in base al nome del parametro.|
