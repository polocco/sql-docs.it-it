# <a name="clearwarnings-method-sqlserverresultset"></a>Metodo clearWarnings (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Cancella tutti gli avvisi segnalati su questo oggetto [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md).  
  
> [!NOTE]  
>  Questo metodo non è attualmente implementato da [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]. Se viene chiamato, restituirà sempre Null.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public void clearWarnings()  
```  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo clearWarnings viene specificato dal metodo clearWarnings nell'interfaccia java.sql.ResultSet.
