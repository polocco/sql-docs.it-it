# <a name="getreference-method-sqlserverxadatasource"></a>Metodo getReference (SQLServerXADataSource)

  Restituisce un riferimento a questo oggetto [SQLServerXADataSource](../../../connect/jdbc/reference/sqlserverxadatasource-class.md).  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public javax.naming.Reference getReference()  
```
