# <a name="supportssubqueriesinquantifieds-method-sqlserverdatabasemetadata"></a>Metodo supportsSubqueriesInQuantifieds (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera un valore che indica se il database supporta le sottoquery nelle espressioni quantificate.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public boolean supportsSubqueriesInQuantifieds()  
```  
  
## <a name="return-value"></a>Valore restituito  
 **true** se supportato. In caso contrario, **false**.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo supportsSubqueriesInQuantifieds viene specificato dal metodo supportsSubqueriesInQuantifieds nell'interfaccia java.sql.DatabaseMetaData.
