# <a name="sqlserversavepoint-constructor-sqlserverconnection-javalangstringname"></a>Costruttore SQLServerSavepoint (SQLServerConnection, java.lang.StringName)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Inizializza una nuova istanza della classe [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) in base alla connessione e al nome specificati.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public SQLServerSavepoint(SQLServerConnection con,  
                          java.lang.String sName)  
```  
  
#### <a name="parameters"></a>Parametri  
 *con*  
  
 Oggetto [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md).  
  
 *sName*  
  
 Valore **String** contenente il nome del punto di salvataggio.
