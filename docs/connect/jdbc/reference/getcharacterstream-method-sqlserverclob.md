# <a name="getcharacterstream-method-sqlserverclob"></a>Metodo getCharacterStream (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore **Clob** designato dall'oggetto java.sql.Clob come oggetto Java.io.Reader.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[Metodo getCharacterStream &#40;&#41;](../../../connect/jdbc/reference/getcharacterstream-method.md)|Restituisce i dati Clob come oggetto java.io.Reader o come flusso di caratteri.|  
|[Metodo getCharacterStream &#40;long, long&#41;](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Restituisce i dati CLOB come oggetto java.io.Reader o come flusso di caratteri con la posizione e la lunghezza specificate.|
