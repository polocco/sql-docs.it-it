# <a name="getmaxcharliterallength-method-sqlserverdatabasemetadata"></a>Metodo getMaxCharLiteralLength (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il numero massimo di caratteri consentito dal database in un valore letterale carattere.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public int getMaxCharLiteralLength()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Valore **int** che indica il numero massimo di caratteri consentito.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo getMaxCharLiteralLength viene specificato dal metodo getMaxCharLiteralLength nell'interfaccia java.sql.DatabaseMetaData.
