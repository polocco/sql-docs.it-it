# <a name="geturl-method-sqlservercallablestatement"></a>Metodo getURL (SQLServerCallableStatement)

  Recupera il valore del parametro designato come oggetto URL nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Descrizione|  
|----------|-----------------|  
|[getURL (int)](../../../connect/jdbc/reference/geturl-method-int.md)|Recupera il valore del parametro designato come oggetto URL nel linguaggio di programmazione Java in base all'indice del parametro.|  
|[getURL (java.lang.String)](../../../connect/jdbc/reference/geturl-method-java-lang-string.md)|Recupera il valore del parametro designato come oggetto URL nel linguaggio di programmazione Java in base al nome del parametro.|
