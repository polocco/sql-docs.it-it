# <a name="getdescription-method-sqlserverdatasource"></a>Metodo getDescription (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce una descrizione dell'origine dati.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.lang.String getDescription()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Valore **String** contenente la descrizione dell'origine dati oppure Null se non viene impostato alcun valore.
