# <a name="getncharacterstream-method-sqlservercallablestatement"></a>Metodo getNCharacterStream (SQLServerCallableStatement)

|----------|-----------------|  
|[Metodo getNCharacterStream &#40;int&#41;](../../../connect/jdbc/reference/getncharacterstream-method-int.md)|Recupera il valore del parametro designato come oggetto java.io.Reader in base all'indice del parametro.|
