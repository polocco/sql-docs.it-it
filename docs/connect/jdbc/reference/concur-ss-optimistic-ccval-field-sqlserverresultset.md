# <a name="concur_ss_optimistic_ccval-field-sqlserverresultset"></a>Campo CONCUR_SS_OPTIMISTIC_CCVAL (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Usato per specificare un tipo di concorrenza ottimistica di lettura/scrittura di [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] senza blocchi di riga.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CCVAL  
```  
  
## <a name="field-value"></a>Valore del campo  
 Valore **int** 1010.
