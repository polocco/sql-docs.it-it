# <a name="isnamed-method-sqlserversavepoint"></a>Metodo isNamed (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Verifica se il punto di salvataggio è stato denominato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public boolean isNamed()  
```  
  
## <a name="return-value"></a>Valore restituito  
 **true** se il punto di salvataggio è denominato. In caso contrario, **false**.
