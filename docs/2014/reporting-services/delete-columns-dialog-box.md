# <a name="delete-columns-dialog-box"></a>Finestra di dialogo Elimina colonne
  Utilizzare la finestra di dialogo **Elimina colonne** per eliminare colonne da un'area dati tabella, matrice o elenco. Per le colonne associate a gruppi, Ã¨ possibile eliminare anche la definizione di gruppo.  
  
## <a name="options"></a>Opzioni  
 **Elimina il gruppo e le relative righe e colonne**  
 Scegliere questa opzione per eliminare le colonne e la definizione di gruppo associata.  
  
 **Elimina solo colonne**
