# <a name="complete---complete-image"></a>Operazione completata - Completamento immagine
  Utilizzare la pagina **Operazione completata** per verificare che la configurazione dell'istanza predisposta sia completata.  
  
## <a name="options"></a>Opzioni  
 Questa finestra di dialogo include un collegamento al file di log del programma di installazione per questa operazione. Per completare l'installazione, fare clic su **Chiudi**.
