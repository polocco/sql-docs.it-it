# <a name="summary-writing-transact-sql"></a>Riepilogo: Scrittura di codice Transact-SQL
  L'editor di query è un passaggio significativo per un ambiente di creazione più [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] sofisticato nello sviluppo. È possibile passare facilmente alla modalità schermo intero dell'editor di query per visualizzare il codice o attivare rapidamente la connessione su un altro server. È possibile cercare funzionalità aggiuntive nella finestra di dialogo **Opzioni** , disponibile nel menu **strumenti** .
