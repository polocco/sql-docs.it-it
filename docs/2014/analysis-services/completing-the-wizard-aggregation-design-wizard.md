# <a name="completing-the-wizard-aggregation-design-wizard"></a>Completamento procedura guidata (Progettazione guidata aggregazioni)
  Utilizzare la pagina **Completamento procedura guidata** per salvare la progettazione delle aggregazioni nelle partizioni selezionate. Ãˆ facoltativamente possibile, inoltre, distribuire ed elaborare le partizioni selezionate al termine delle operazioni di progettazione e di salvataggio delle aggregazioni in Progettazione guidata aggregazioni.  
  
## <a name="options"></a>Opzioni  
 **Nome**  
 Immettere un nome per la progettazione aggregazioni.  
  
 **Distribuisci ed elabora aggregazioni ora**  
 Consente di completare la procedura guidata, assegnare la progettazione aggregazioni alle partizioni selezionate e quindi distribuire ed elaborare queste ultime.  
  
 **Salva le aggregazioni senza elaborarle**  
 Consente di completare la procedura guidata e assegnare la progettazione aggregazioni alle partizioni selezionate, ma non di distribuire ed elaborare queste ultime.
