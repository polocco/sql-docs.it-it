# <a name="column-width-dialog-box-ssas"></a>Finestra di dialogo Larghezza colonna (SSAS)
  La finestra di dialogo **Larghezza colonna** consente di specificare la larghezza di una colonna in Progettazione modelli. Per accedere a questa finestra di dialogo, fare clic con il pulsante destro del mouse in una colonna, quindi scegliere **Larghezza colonna**.
