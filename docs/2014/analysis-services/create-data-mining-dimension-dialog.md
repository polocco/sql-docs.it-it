# <a name="create-data-mining-dimension-dialog"></a>Finestra di dialogo Crea dimensione di data mining
  Utilizzare la finestra di dialogo **Crea dimensione di data mining** per creare una nuova dimensione di data mining in base a un modello di data mining OLAP.  
  
## <a name="options"></a>Opzioni  
 **Nome modello**  
 Consente di selezionare il modello di data mining OLAP su cui basare la dimensione di data mining nell'elenco dei modelli di data mining OLAP nel progetto di [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] .  
  
 **Nome dimensione**  
 Consente di digitare un nome per la nuova dimensione.  
  
 **Crea cubo**  
 Selezionare questa opzione per creare un cubo basato sulla dimensione del modello di data mining da creare. Dopo aver selezionato questa opzione, Ã¨ possibile digitare un nuovo nome per il cubo.
