# <a name="mssqlserver_17147"></a>MSSQLSERVER_17147
    
## <a name="details"></a>Dettagli  
  
|Attributo|valore|  
|-|-|  
|Nome prodotto|SQL Server|  
|ID evento|17147|  
|Origine evento|MSSQLSERVER|  
|Componente|SQLEngine|  
|Nome simbolico|STARTUP_SERVERKILLED|  
|Testo del messaggio|SQL Server verrà terminato a causa dell'arresto del sistema. Questo è un messaggio informativo. Non è richiesta alcuna azione da parte dell'utente.|
