# <a name="windowsserviceidentityconfigured-property"></a>WindowsServiceIdentityConfigured - proprietà
  Restituisce l'ultima identità con cui il servizio Windows ReportServer è stato configurato per l'esecuzione. Di sola lettura.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
Public Dim WindowsServiceIdentityConfigured As String  
```  
  
```csharp  
public string WindowsServiceIdentityConfigured;  
```  
  
## <a name="property-values"></a>Valori della proprietà  
 Valore **String** che contiene l'ultima identità con cui il servizio Windows ReportServer è stato configurato per l'esecuzione.
