# <a name="kpiweight-mdx"></a>KPIWeight (MDX)


  Restituisce il peso dell'indicatore di prestazioni chiave (KPI) specificato.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
KPIWeight(KPI_Name)  
```  
  
## <a name="arguments"></a>Argomenti  
 *KPI_Name*  
 Espressione stringa valida che specifica il nome dell'indicatore di prestazioni chiave (KPI).  
  
## <a name="remarks"></a>Osservazioni  
 Il valore restituito rappresenta il contributo dell'indicatore KPI all'elemento padre.
