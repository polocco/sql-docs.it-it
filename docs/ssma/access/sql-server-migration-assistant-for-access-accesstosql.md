# <a name="sql-server-migration-assistant-for-access-accesstosql"></a>SQL Server Migration Assistant per Access (AccessToSQL)

[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Migration Assistant (SSMA) per l'accesso è uno strumento per la migrazione [!INCLUDE[msCoName](../../includes/msconame_md.md)] dei database da versioni di Access [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] da 97 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] a 2010 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] a 2012 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] , 2014, 2016, 2017 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] in Windows e Linux, 2019 in [!INCLUDE[msCoName](../../includes/msconame_md.md)] Windows e Linux o nel database SQL di Azure. SSMA per Access converte gli oggetti di database [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] di Access negli oggetti di database SQL di Azure o [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] , li carica nel database SQL di Azure e quindi esegue la migrazione [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] dei dati dall'accesso al database SQL di Azure o.
  
Questa documentazione presenta SSMA per l'accesso e fornisce istruzioni dettagliate per la migrazione dei database di Access a o del [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] database SQL di Azure e informazioni sui problemi che potrebbero verificarsi dopo la migrazione.
