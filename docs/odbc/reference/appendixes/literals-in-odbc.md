# <a name="literals-in-odbc"></a>Valori letterali in ODBC
La sintassi nelle sezioni seguenti viene utilizzata per i valori letterali interval e numerici in ODBC. Questa sintassi viene fornita qui come supporto per i writer di driver quando le conversioni vengono eseguite da un tipo di stringa di caratteri a un tipo numerico o di intervallo o da un tipo numerico o di intervallo a un tipo di stringa di caratteri. Per altre informazioni, vedere [valori letterali di intervallo](../../../odbc/reference/appendixes/interval-literals.md) e [valori letterali numerici](../../../odbc/reference/appendixes/numeric-literals.md) nell'appendice D: tipi di dati.  
  
 Questa appendice contiene i seguenti argomenti.  
  
-   [Sintassi dei valori letterali intervallo](../../../odbc/reference/appendixes/interval-literal-syntax.md)  
  
-   [Sintassi dei valori letterali numerici](../../../odbc/reference/appendixes/numeric-literal-syntax.md)
