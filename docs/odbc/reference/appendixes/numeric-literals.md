# <a name="numeric-literals"></a>Valori letterali numerici
I valori letterali numerici vengono utilizzati quando i valori di dati numerici vengono archiviati in stringhe di caratteri. Per la conversione di dati SQL numerici in una stringa di SQL_C_CHAR o la conversione di dati Numeric C in una stringa SQL_CHAR o SQL_VARCHAR, la sintassi letterale numerica viene utilizzata per specificare gli elementi archiviati nella destinazione. Per la conversione di un valore numerico archiviato come stringa SQL_C_CHAR in dati SQL numerici o come stringa di SQL_CHAR in dati Numerici C, questa sintassi viene utilizzata per convalidare ci√≤ che viene archiviato nell'origine.  
  
 I valori letterali numerici devono essere conformi alla sintassi definita nella sezione [Sintassi dei valori letterali numerici](../../../odbc/reference/appendixes/numeric-literal-syntax.md) nell'Appendice C: Grammatica SQL.
