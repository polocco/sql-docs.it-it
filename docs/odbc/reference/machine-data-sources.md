# <a name="machine-data-sources"></a>Origini dati del computer
Le *origini dati del computer* vengono archiviate nel sistema con un nome definito dall'utente. Associato al nome dell'origine dati è costituito da tutte le informazioni necessarie a gestione driver e driver per connettersi all'origine dati. Per un'origine dati Xbase, può corrispondere al nome del driver Xbase, al percorso completo della directory contenente i file Xbase e ad alcune opzioni che indicano al driver come utilizzare tali file, ad esempio la modalità utente singolo o di sola lettura. Per un'origine dati Oracle, potrebbe essere il nome del driver Oracle, il server in cui risiede il sistema DBMS Oracle, la stringa di connessione SQL * NET che identifica il driver di \* rete SQL da utilizzare e l'ID di sistema del database nel server.
