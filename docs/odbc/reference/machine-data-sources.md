# <a name="machine-data-sources"></a>Origini dati del computer
*Le origini dati macchina* vengono archiviate nel sistema con un nome definito dall'utente. Associato al nome dell'origine dati sono tutte le informazioni necessarie a Gestione Driver e al driver per connettersi all'origine dati. Per un'origine dati Xbase, questo potrebbe essere il nome del driver Xbase, il percorso completo della directory contenente i file Xbase e alcune opzioni che indicano al driver come utilizzare tali file, ad esempio la modalit√† utente singolo o di sola lettura. Per un'origine dati Oracle, questo potrebbe essere il nome del driver Oracle, il server in cui\*risiede il sistema DBMS Oracle, la stringa di connessione SQL-Net che identifica il driver di rete SQL da utilizzare e l'ID di sistema del database nel server.
