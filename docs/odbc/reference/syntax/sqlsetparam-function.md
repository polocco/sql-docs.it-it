# <a name="sqlsetparam-function"></a>Funzione SQLSetParam
**Conformità**  
 Versione introdotta: ODBC 1.0 Standards Compliance: Deprecated  
  
 **Riepilogo**  
 In ODBC 2.0 la funzione ODBC 1.0 **SQLSetParam** è stata sostituita da **SQLBindParameter.** Per ulteriori informazioni, vedere [Funzione SQLBindParameter](../../../odbc/reference/syntax/sqlbindparameter-function.md).  
  
> [!NOTE]  
>  Per ulteriori informazioni su ciò che Gestione Driver esegue il mapping di questa funzione a quando un ODBC 2. *l'applicazione x* funziona con un'applicazione ODBC 3. *x,* vedere [Mapping di funzioni obsolete](../../../odbc/reference/appendixes/mapping-deprecated-functions.md).
