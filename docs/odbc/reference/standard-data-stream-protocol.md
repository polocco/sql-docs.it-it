# <a name="standard-data-stream-protocol"></a>Protocollo del flusso di dati standard
Un protocollo di flusso di dati standard è un modo per accedere ai dati in DBS eterogenei. Infatti, esiste già un protocollo di flusso di dati standard:

- Lo standard RDA (Remote Database Access) ANSI/ISO: [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). Anche se il sistema ANSI/ISO promette, non è ampiamente implementato oggi.
