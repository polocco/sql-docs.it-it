# <a name="odbc-header-files"></a>File di intestazione ODBC
> [!NOTE]  
>  La ridistribuzione dei file di intestazione ODBC non è necessaria. Nessuno è considerato "Core".  
  
 **File di intestazione**  
  
 Odbcinst.h  
  
 Sql.h (informazioni in lingua inglese)  
  
 Sqlext.h (informazioni in lingua inglese)  
  
 Sqltypes.h  
  
 Sqlucode.h  
  
 File Msdasql.h  
  
 Msdadc.h  
  
 Sqlspi.h (utilizzato per lo sviluppo di driver, vedere [ODBC Service Provider Interface (SPI) Riferimento](../../../odbc/reference/syntax/odbc-service-provider-interface-spi-reference.md) per ulteriori informazioni) disponibile a partire da Windows 8 SDK.
