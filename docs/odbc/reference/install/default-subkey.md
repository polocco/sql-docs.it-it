# <a name="default-subkey"></a>Sottochiave Default
Il Registro di sistema può specificare un'origine dati predefinita con la sottochiave Default.The registry may specify a default data source with the Default subkey. Questa sottochiave è un caso speciale di una sottochiave di specifica dell'origine dati e ha gli stessi valori di qualsiasi altra sottochiave di specifica dell'origine dati. L'unica differenza è che non è elencato come un valore nella sottochiave origini dati ODBC.
