# <a name="default-subkey"></a>Sottochiave Default
Il registro di sistema può specificare un'origine dati predefinita con la sottochiave predefinita. Questa sottochiave è un caso speciale di una sottochiave specifica dell'origine dati e ha gli stessi valori di qualsiasi altra sottochiave specifica dell'origine dati. L'unica differenza è che non è elencato come valore nella sottochiave origini dati ODBC.
