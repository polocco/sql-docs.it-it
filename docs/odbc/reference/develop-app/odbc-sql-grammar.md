# <a name="odbc-sql-grammar"></a>Grammatica SQL ODBC
Le applicazioni interoperabili devono sempre utilizzare la grammatica SQL ODBC nelle istruzioni SQL. Tuttavia, una notevole variazione è possibile anche all'interno di questa grammatica. Per ulteriori informazioni, vedere [Interoperabilità delle istruzioni SQL](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md).
