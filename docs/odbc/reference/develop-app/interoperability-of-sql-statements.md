# <a name="interoperability-of-sql-statements"></a>Interoperabilità delle istruzioni SQL
Come il resto di un'applicazione, le istruzioni SQL possono essere interoperabili o specifiche di DBMS. E come il resto dell'applicazione, la scelta di come le istruzioni SQL interoperabili devono essere dipende dal tipo di applicazione. Le applicazioni personalizzate hanno meno probabilità di utilizzare istruzioni SQL interoperabili perché sono in genere progettate per sfruttare le funzionalità di uno o forse due DBS. Le applicazioni generiche utilizzano istruzioni SQL interoperabili perché sono progettate per funzionare con un'ampia gamma di DBSMO. E le applicazioni verticali di solito cadono da qualche parte nel mezzo, richiedendo un certo livello di funzionalità, ma in caso contrario utilizzando istruzioni SQL interoperabili.
