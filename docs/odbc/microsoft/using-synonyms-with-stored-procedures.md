# <a name="using-synonyms-with-stored-procedures"></a>Uso dei sinonimi con le stored procedure
> [!IMPORTANT]  
>  Questa funzionalità verrà rimossa in una versione futura di Windows. Evitare di usare questa funzionalità in un nuovo progetto di sviluppo e prevedere interventi di modifica nelle applicazioni in cui è attualmente implementata. Utilizzare invece il driver ODBC fornito da Oracle.  
  
 Il Driver Microsoft ODBC per Oracle versioni 2.0 e 2.5 non supporta nodi durante la chiamata di stored procedure Oracle. I sinonimi funzionano come previsto se utilizzati con altri oggetti di database Oracle, ad esempio tabelle.
