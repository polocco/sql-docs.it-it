# <a name="encryption-access-driver"></a>Crittografia (driver Access)
(Questo controllo è disponibile solo quando la versione 4. *X* formato è selezionato.)  
  
 Con il rilascio della versione 4.0 del modulo di gestione di database Microsoft Jet, la crittografia del database è consentita durante la creazione di database di Access. Per ulteriori informazioni, vedere Jet *Database Engine Programmer's Reference (informazioni* in linea di base).
