# <a name="drop-index-statement"></a>Istruzione DROP INDEX
Quando si utilizza il driver di Microsoft Access, dBASE o Paradox, la sintassi dell'istruzione DROP INDEX è "DROP INDEX a on b" dove "a" è il nome dell'indice e "b" è il nome della tabella (non DROP *INDEX-name*).  
  
 Quando viene utilizzato il driver Paradox, l'istruzione DROP INDEX elimina i file di indice secondari o secondari di Paradox.  
  
 L'istruzione DROP INDEX non è supportata per i driver di Microsoft Excel o di testo.
