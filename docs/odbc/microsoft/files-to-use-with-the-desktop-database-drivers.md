# <a name="files-to-use-with-the-desktop-database-drivers"></a>File da usare con i driver di database desktop
I driver di database desktop ODBC vengono utilizzati con Microsoft Jet motore di database, i file ISAM installabili e i file Microsoft ODBC. Non tutti sono necessari per la ridistribuzione del prodotto e non tutti sono componenti di base.  
  
|Per informazioni su|Vedere|  
|---------------------------|---------|  
|File usati da ogni driver di database desktop|File del driver ODBC|  
|File Microsoft Jet usati da ogni driver di database desktop|File Microsoft Jet|  
|File di driver IISAM che forniscono l'accesso ai dati tra Microsoft Jet e l'altro sistema DBMS|File del driver ISAM installabile|
