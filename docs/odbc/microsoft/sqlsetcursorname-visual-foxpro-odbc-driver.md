# <a name="sqlsetcursorname-visual-foxpro-odbc-driver"></a>SQLSetCursorName (driver ODBC Visual FoxPro)
> [!NOTE]  
>  In questo argomento sono contenute informazioni specifiche del driver ODBC di Visual FoxPro. Per informazioni generali su questa funzione, vedere l'argomento appropriato in [Riferimento all'API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Supporto: Completo  
  
 Conformità API ODBC: livello di baseODBC API Conformance: Core Level  
  
 Associa un nome di cursore a un handle di istruzione attivo, *hstmt*. **SQLSetCursorName** è incluso nell'API driver ODBC di Visual FoxPro perché fa parte della funzionalità dell'API ODBC di livello base; non può essere utilizzato con altre funzioni API perché il driver non supporta gli aggiornamenti posizionati.  
  
 Per ulteriori informazioni, vedere [SQLSetCursorName](../../odbc/reference/syntax/sqlsetcursorname-function.md) in *ODBC Programmer's Reference*.
