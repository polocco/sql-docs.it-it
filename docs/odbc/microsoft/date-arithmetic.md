# <a name="date-arithmetic"></a>Aritmetica delle date
Il driver supporta l'aggiunta e la sottrazione di un numero intero da una colonna DATE, TIME o TIMESTAMP. Per una colonna DATE, il numero intero specifica il numero di giorni da aggiungere o sottrarre. Per una colonna TIME o TIMESTAMP, il numero intero specifica il numero di secondi da aggiungere o sottrarre.
