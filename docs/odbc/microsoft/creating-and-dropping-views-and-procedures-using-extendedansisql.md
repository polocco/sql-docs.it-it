# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Creazione ed eliminazione di viste e procedure tramite ExtendedAnsiSQL
Quando il flag ExtendedAnsiSQL è attivato, Jet 4,0 supporta la possibilità di creare ed eliminare viste e procedure tramite SQL. Queste viste e procedure non saranno visibili tramite l'accesso.
