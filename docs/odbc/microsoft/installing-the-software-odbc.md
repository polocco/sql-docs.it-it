# <a name="installing-the-software-odbc"></a>Installazione del software (ODBC)
> [!IMPORTANT]  
>  Questa funzionalità verrà rimossa in una versione futura di Windows. Evitare di usare questa funzionalità in un nuovo progetto di sviluppo e prevedere interventi di modifica nelle applicazioni in cui è attualmente implementata. Utilizzare invece il driver ODBC fornito da Oracle.  
  
 Il driver ODBC per Oracle è uno dei componenti di accesso ai dati. Accompagna altri componenti ODBC, ad esempio L'Amministratore origine dati ODBC, e deve essere già installato. Il driver è disponibile anche in "Driver e altri download" nel sito Web Online del Servizio Supporto Tecnico Clienti Microsoft [all'www.microsoft.com](https://www.microsoft.com).  
  
 Il software di rete deve essere installato in base alla propria documentazione. Il driver ODBC per Oracle non richiede particolari considerazioni di installazione, purché il software di rete sia supportato.  
  
 Il software Oracle deve essere installato in base alla propria documentazione. Il driver ODBC per Oracle in genere non richiede particolari considerazioni di installazione, purché il driver supporta la versione. Tuttavia, per mantenere i prodotti compatibili, installare l'ultimo driver ODBC per Oracle per assicurarsi di disporre della versione più recente del driver. Oracle gestisce un sito FTP pubblico in cui invia, tra le altre cose, patch ai prodotti server Oracle e al componente client fornito con i prodotti server. Queste patch sono necessarie per il corretto funzionamento di diversi prodotti e tecnologie Microsoft. Per ulteriori informazioni su questo sito, vedere [Oracle Software Patches](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  L'installazione del software Oracle su MDAC/Windows DAC potrebbe sovrascrivere le versioni correnti di MDAC. Se si verificano problemi con i componenti ODBC, reinstallare MDAC.
