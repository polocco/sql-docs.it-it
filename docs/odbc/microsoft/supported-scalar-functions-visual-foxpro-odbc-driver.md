# <a name="supported-scalar-functions-visual-foxpro-odbc-driver"></a>Funzioni scalari supportate (driver ODBC Visual FoxPro)
Il driver ODBC Visual FoxPro supporta tre tipi di funzioni scalari, come definito nella grammatica SQL ODBC: stringhe, numeriche e funzioni di data e ora.  
  
 In questa sezione vengono trattati gli argomenti seguenti.  
  
-   [Funzioni per i valori stringa (driver ODBC Visual FoxPro)](../../odbc/microsoft/string-functions-visual-foxpro-odbc-driver.md)  
  
-   [Funzioni numeriche (driver ODBC Visual FoxPro)](../../odbc/microsoft/numeric-functions-visual-foxpro-odbc-driver.md)  
  
-   [Funzioni data e ora (driver ODBC Visual FoxPro)](../../odbc/microsoft/time-and-date-functions-visual-foxpro-odbc-driver.md)
