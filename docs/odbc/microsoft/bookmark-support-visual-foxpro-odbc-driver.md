# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>Supporto dei segnalibri (driver ODBC Visual FoxPro)
Il driver ODBC di Visual FoxPro supporta segnalibri semplici. Quando si chiama [SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md) con il SQL_BOOKMARK_PERSISTENCE *InfoType*, il valore restituito viene SQL_BP_SCROLL.  
  
 Per ulteriori informazioni sui segnalibri, vedere [Segnalibri (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md).
