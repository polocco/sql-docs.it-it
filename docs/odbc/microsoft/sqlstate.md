# <a name="sqlstate"></a>SQLSTATE
Per gli errori che si verificano nell'origine dati, il driver ODBC esegue il mapping dell'errore nativo restituito al SQLSTATE appropriato. Per gli errori rilevati dal driver o da Gestione Driver, il driver ODBC o Gestione Driver genera il SQLSTATE appropriato.
