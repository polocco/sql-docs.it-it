# <a name="api-functions-odbc-driver-for-oracle"></a>Funzioni API (driver ODBC per Oracle)
> [!IMPORTANT]  
>  Questa funzionalità verrà rimossa in una versione futura di Windows. Evitare di usare questa funzionalità in un nuovo progetto di sviluppo e prevedere interventi di modifica nelle applicazioni in cui è attualmente implementata. Utilizzare invece il driver ODBC fornito da Oracle.  
  
 Il driver Microsoft ODBC per Oracle supporta le funzioni API Core Level, Level 1 e Level 2. Queste funzioni sono elencate in Livelli di [conformità ODBC](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md).  
  
-   La conformità CLI [(Core Level Interface)](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md) fornisce funzionalità definite nella specifica ISO CLI e le funzionalità obbligatorie definite nella specifica Open Group CLI.  
  
-   [La](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md) conformità di livello 1 fornisce funzionalità dell'interfaccia di livello base, nonché funzionalità aggiuntive, ad esempio le transazioni.  
  
-   [La](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md) conformità di livello 2 fornisce funzionalità di livello 1, nonché funzionalità aggiuntive, ad esempio segnalibri, parametri dinamici ed esecuzione asincrona di funzioni ODBC.
