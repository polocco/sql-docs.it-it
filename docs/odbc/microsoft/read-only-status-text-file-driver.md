# <a name="read-only-status-text-file-driver"></a>Stato di sola lettura (driver file di testo)
Quando viene utilizzato il driver di testo, le tabelle dell'origine dati sono di sola lettura. Tuttavia, le applicazioni possono eseguire inserimenti per le tabelle di testo.
