# <a name="sqlspecialcolumns-desktop-database-drivers"></a>SQLSpecialColumns (driver di database desktop)
Verrà restituito un indice univoco (se presente) per il flag SQL_BEST_ROWID in *fColType*. Non verrà restituito alcun set di risultati per il flag di SQL_ROWVER.  
  
 Tutti gli ID di riga hanno un ambito di SQL_SCOPE_CURROW.  
  
 La corrispondenza dei modelli non è supportata per l'argomento *szTableQualifier* o *szTableName.*
